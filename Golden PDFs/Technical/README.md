# Technical

PDFs chosen for structure, layout, and PDF-specific features rather than content.

**Examples to add here:**

- Multi-page documents with consistent and varying page sizes
- Portrait and landscape pages in one file
- Pre-rotated page metadata
- Text-only, vector-only, and image-heavy pages
- Embedded fonts and selectable text layers
- Documents with bookmarks, links, or form fields (if supported)

Use these to test rendering accuracy, export fidelity, and text preservation.
