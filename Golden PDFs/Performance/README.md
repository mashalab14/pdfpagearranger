# Performance

PDFs used to measure speed, memory use, and responsiveness.

**Examples to add here:**

- High page counts (50+, 100+, 500+ pages)
- Large file sizes (10 MB, 50 MB, 100 MB+)
- Pages with dense vector art or high-resolution images
- Documents that stress thumbnail generation and export

Use these for manual timing checks and future performance regression tests. Avoid checking very large binaries into git unless necessary; document expected file sizes in commit messages or test notes.
