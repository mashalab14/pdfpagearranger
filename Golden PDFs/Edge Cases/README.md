# Edge Cases

PDFs that expose bugs in import, preview, editing, or export under unusual conditions.

**Examples to add here:**

- Password-protected or encrypted files
- Single-page and very large page-count documents
- Extreme aspect ratios (very wide or tall pages)
- Blank or nearly blank pages
- Mixed DPI scans and low-quality inputs
- Files with unusual metadata, rotation, or crop boxes
- Borderline-valid PDFs that still open in Preview or Acrobat

Use these to harden error handling and prevent regressions on fragile inputs.
